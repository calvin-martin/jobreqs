# QA Requirements - Claude Template

This file contains QA-specific adaptations of the general requirements template, enhanced with insights from successful QA lead guidance and interactive HTML implementation.

## QA-Specific Categories (Enhanced Framework)

### Core Categories for All Levels:
1. **Requirement Analysis**: Understanding and analyzing user stories, acceptance criteria, ambiguity detection
2. **Test Design**: Creating test cases, test scenarios, test data, and comprehensive strategies
3. **Test Execution**: Running tests, documenting results, managing test environments, regression testing
4. **Defect Management**: Finding, documenting, tracking bugs through lifecycle, prevention strategies
5. **API & Technical Testing**: Understanding developer tools, API testing basics, database validation
6. **Domain Knowledge**: Understanding application logic, business rules, testing tools, business processes
7. **Automation & Leadership**: (Senior levels) Test automation, team mentoring, strategic planning

### Level-Specific Category Adaptations:

**Level 1 Additional Categories:**
- **Identify Testable Features**: Distinguishing functional vs non-functional requirements, positive vs negative testing
- **Execute Complex Test Cases**: Environment awareness, test execution flow, result comparison

**Level 2+ Enhancements:**
- **Advanced Test Design**: Decision tables, pairwise testing, use case testing
- **Cross-Platform Testing**: Browser compatibility, mobile responsive testing
- **Performance Awareness**: Basic performance concepts, monitoring, load testing
- **Process Knowledge**: Agile testing, estimation techniques, risk assessment

**Level 3+ Specializations:**
- **Specialized Testing Areas**: Security, accessibility, usability, compatibility testing
- **Advanced API & Integration**: Microservices, contract testing, integration strategies
- **Process Improvement**: Quality metrics, continuous testing, process optimization

## QA Level Progression

- **Level 1**: Trainee → Junior QA Engineer (Foundation skills, 30-35 specific requirements)
- **Level 2**: Junior → QA Engineer (Advanced techniques, 15-20 requirements)
- **Level 3**: QA Engineer → Senior QA Engineer (Strategy + automation intro, 20-25 requirements)
- **Level 4**: Senior → Lead QA Engineer (Advanced automation + leadership, 20-25 requirements)
- **Level 5**: Lead → Principal QA Engineer (Strategic vision + enterprise architecture, 15-20 requirements)
