# Engineering Requirements - Claude Template

This file contains Engineering-specific adaptations of the general requirements template.

## Engineering-Specific Categories

1. **Programming Fundamentals**: Core programming concepts, algorithms, data structures
2. **System Design & Architecture**: Scalability, performance, design patterns
3. **Development Methodologies**: Agile, DevOps, CI/CD, testing practices
4. **Technical Tools & Platforms**: IDEs, frameworks, cloud platforms, databases
5. **Problem Solving & Debugging**: Troubleshooting, optimization, code review
6. **Collaboration & Documentation**: Code documentation, technical communication
7. **Technical Leadership**: (Senior levels) Architecture decisions, team mentoring

## Engineering Level Progression

- **Level 1**: Engineering Trainee → Junior Developer
- **Level 2**: Junior → Software Engineer
- **Level 3**: Software Engineer → Senior Software Engineer
- **Level 4**: Senior → Lead Engineer/Tech Lead
- **Level 5**: Lead → Principal Engineer/Engineering Manager

## Reference Quality Standards

For Engineering requirements, prioritize:
- **Code tutorials** with working examples and repositories
- **Documentation** with practical implementation guides
- **Interactive coding platforms** (GitHub, CodePen, etc.)
- **Architecture guides** with real-world system examples
- **Best practices** with before/after code comparisons
